## GD_WEB

### 사용 기술스택

- JavaScript

- ECMAScript (ES6)

- React.js

- React-Query

  - 백엔드와의 통신에 사용

- Tailwind

  - CSS 스타일링에 사용

- React-Bootstrap

  - 스타일링 된 컴포넌트를 load
